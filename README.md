QontMan
======

**QontMan это телефонная книга с возможностью отображения древовидной структуры взаимосвязей контактов.**

Возможности:
- сохранение и чтение настроек расположения окон
- контактная информация храниться в базе данных
- изображения контактов хранятся в базе данных
- различные возможности фильтрации
- древовидная структура контактов
- построение диаграмм подчиненности
- импорт и экспорт контактов в vcf файл
- возможность масштабирования программы для совместного использования

----

Программа написана с использованием QT, для сборки не требуются дополнительные библиотеки, требуется собранный плагин для поддержки FireBird.
